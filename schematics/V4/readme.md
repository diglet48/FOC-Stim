You can order assembled boards from JLCPCB

## Mainboard
* Single-sided assembly (PCBA type standard, not economic)
* Select lead-free HASL finish.
* Verify the rotation of all chips and diodes against the reference image. The initial placement usually has U11 oriented wrongly. 

## Frontpanel
* Single-sided assembly (PCBA type economic) OR no assembly
* Select lead-free HASL finish.

You can choose to let JLCPCB assemble the SMD nuts and 2.0mm headers,
the error rate for letting users solder these components at home is
surprisingly high.
